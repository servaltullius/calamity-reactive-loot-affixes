## 자동 합성형 룬워드 (78개) — 스타일 매핑 기반

기존의 “무기=적응형 참격 / 방어구=생존 버프” 단일 템플릿 대신,  
이제는 **룬워드 ID 우선 매핑 + 베이스 타입 해시 분산**으로 합성됩니다.

### 자동 합성 핵심 규칙

1. **우선 매핑(핵심 룬워드)**  
   - 룬워드 테마에 맞춰 스타일을 고정 배정합니다.  
   - 예: `Malice/Plague` → 독 블룸, `Lawbringer/Wrath` → 저주형 디버프

2. **자동 분산(나머지 룬워드)**  
   - 권장 베이스 타입(무기/방어구)과 룬워드 ID 해시로 스타일을 분산해,  
     특정 효과(예: 적응형 참격)만 반복되는 현상을 줄입니다.

3. **안전 폴백**  
   - 특정 스펠 로드 실패 시, 기존 레거시 템플릿으로 폴백해 제작 실패를 방지합니다.

### Adaptive + modeCycle 정책 (중요)

1. `CastSpellAdaptiveElement`에 `modeCycle`이 있으면 런타임에서 `manualOnly=true`를 강제합니다.  
2. 수동 모드 인덱스는 `0=Auto(적응형 자동 선택)`이며, `1..N`이 수동 고정 원소/스펠입니다.  
3. 즉, Adaptive 계열은 기본 자동 분석을 유지하면서 필요할 때만 수동 오버라이드가 적용됩니다.

### 자동 합성 스타일 카탈로그

- **Adaptive Strike**  
  - 적중 시 약점 원소 기반 동적 참격 (`CAFF_SPEL_DMG_*_DYNAMIC`)
- **Adaptive Exposure**  
  - 적중 시 강점 원소 노출/파쇄 (`CAFF_SPEL_*_SHRED`)
- **Fire / Frost / Shock Strike**  
  - 단일 원소 참격 (화염/냉기/충격 테마)
- **Poison / Tar / Siphon Bloom**  
  - DoT 블룸 계열 (독 구름 / 둔화 타르 / 자원 재생 약화)
- **Curse (Fragile / Slow Attack / Fear / Frenzy)**  
  - 저주형 디버프 + 군중 제어(공포/광란) 치환
- **Self Haste**  
  - 자가 공속/전투 템포 버프
- **Self Ward / Barrier / Meditation / Phase / Phoenix**  
  - 방어/기동/회복 계열 자가 생존 버프
- **Self Cloak (Flame / Frost / Shock)**  
  - D2 오라 감성을 스카이림 바닐라 Cloak 주문으로 치환
- **Self Flesh (Oak / Stone / Iron / Ebony)**  
  - 방어 오라 계열을 바닐라 Flesh 주문으로 치환
- **Self Muffle / Self Invisibility**  
  - 스카이림 바닐라 은신 계열(소음 감소/투명화) 치환
- **Soul Trap Utility**  
  - 바닐라 Soul Trap 계열 유틸리티 치환

  - 블룸 계열도 룬워드별 proc/ICD/per-target ICD를 개별 조정

  - Soul Trap 치환형의 proc/ICD를 실전 체감 기준으로 상향/차등
- **오라/방어 계열**: Beast / Dragon / Hand of Justice / Flickering Flame / Ice / Pride / Metamorphosis

- 수치(확률/ICD)는 룬 개수(`runeCount`) 기반으로 클램프됩니다.
