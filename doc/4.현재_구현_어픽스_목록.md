# 현재 구현된 어픽스(효과) 목록 — 한국어 요약

> 기준 파일: `Data/SKSE/Plugins/CalamityAffixes/affixes.json` (현재 83개)
>
> 2026-02-08 업데이트: 룬워드는 D2/D2R 기준 94개 카탈로그로 확장되었고,
> `affixes.json`에 없는 룬워드 결과 어픽스는 SKSE 런타임에서 자동 합성됩니다.

이 문서는 “어픽스 이름은 보이는데, 효과 설명(무슨 일이 일어나는지)을 모르면 불편하다”는 문제를 해결하기 위해
현재 구현된 어픽스를 **한국어로 요약**합니다.

## 용어/표기

- **적중(Hit)**: 플레이어가 적에게 피해를 주는 이벤트(무기/마법 등)
- **피격(Incoming Hit)**: 플레이어가 공격을 맞는 이벤트
- **처치(Kill)**: 플레이어(또는 플레이어 소유 소환수)가 적을 처치했을 때의 이벤트
- **DoT 적용(Dot Apply)**: DoT(지속피해) **틱마다가 아니라**, DoT가 “적용/갱신”될 때 1회만 트리거
- **발동 확률**: “몇 % 확률로 효과 발동”
- **ICD(내부 쿨타임)**: 발동 후 같은 효과가 다시 발동하기까지의 최소 대기 시간

## 룬워드(2차 확장, D2/D2R 94개 카탈로그)

- 룬워드 레시피는 Prisma 패널에서 94개 전체를 검색/선택할 수 있습니다.
- 베이스 타입은 레시피별 “권장 타입”만 안내하며, 강제하지 않습니다.
- 완성 시 베이스 인스턴스의 어픽스는 해당 룬워드 최종 어픽스로 교체됩니다.
- 94개 전체 레시피별 효과 설명은 `doc/5.룬워드_94개_효과_정리.md`를 참고하세요.

### 룬워드 효과 적용 방식

- **직접 정의형(기존 16개)**  
  - `runeword_enigma_final`, `runeword_infinity_final`, `runeword_fortitude_final`, `runeword_insight_final` 등  
  - 개별 룬워드별로 수동 튜닝된 효과/수치 사용

- **런타임 자동 합성형(누락분)**  
  - `affixes.json`에 직접 정의되지 않은 룬워드 결과는 제작 실패 대신 자동 합성 템플릿 적용
  - 룬워드 ID 우선 매핑(DoT 블룸/저주/원소 참격/자가 버프/유틸) + 베이스 타입 해시 분산으로 스타일을 배정
  - D2 오라 감성은 스카이림 바닐라 주문으로 치환(Flame/Frost/Shock Cloak, Oak/Stone/Iron/Ebonyflesh, Fear/Frenzy, Muffle/Invisibility/Soul Trap)
  - 주요 룬워드는 proc/ICD/per-target ICD를 개별 튜닝(대표: Lawbringer/Wrath/Kingslayer/Delirium/Nadir/Dragon/Pride 등)
  - 스펠 조회 실패 시 레거시 합성 템플릿으로 폴백하여 제작 실패를 방지

### 참고

- “룬워드 카탈로그(레시피)”와 “어픽스 정의(효과 구현)”는 분리되어 관리됩니다.
- 따라서 레시피가 먼저 확장되어도, 효과는 직접 정의형 + 자동 합성형 조합으로 즉시 동작합니다.
