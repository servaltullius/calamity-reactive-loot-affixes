# v1.2.17 RC 시리즈 통합 패치노트 (rc01~rc70)

- 기준 범위: `v1.2.17-rc01` ~ `v1.2.17-rc70`
- 작성 기준일: 2026-02-21
- 출처: GitHub Release 본문(태그별) + 저장소의 개별 RC 노트 문서
- 목적: RC 전 구간 변경점을 한 문서에서 추적 가능하도록 통합

## 주제별 요약

- MCM/설정 안정화: 초기 RC에서 MCM 중복/소실/정합성 문제를 단계적으로 해소하고, 후반 RC에서 드랍 확률 슬라이더와 외부 설정 영속화(`user_settings.json`)를 안정화했습니다.
- 룬워드/툴팁 UX: 룬워드 패널/툴팁 역할 분리, 즉시 갱신, 이름/설명 로컬라이징, 합성 룬워드 요약 누락 보완을 진행했습니다.
- 드랍 파이프라인: 확률 밸런스 조정과 함께 `SPID + 런타임` 하이브리드 정책을 고정하고, 시체/월드/컨테이너 경로의 중복 롤과 과발동 체감을 정리했습니다.
- UserPatch 체계: 자동 탐지, 위저드 UX, 번들 EXE, ESPFE(ESL flag), 모드 몬스터 드랍 타깃 확장, 로드오더 충돌 복원까지 사용자 환경 적응성을 강화했습니다.
- 런타임 안정성/검증: 포인터 안전 가드, 계약(Contract) 기반 검증/로딩, 린트 스키마 검증 추가, 테스트 모듈화로 릴리즈 회귀 리스크를 줄였습니다.
