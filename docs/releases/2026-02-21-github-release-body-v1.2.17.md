## Calamity Reactive Loot & Affixes v1.2.17

`v1.2.16` 이후 RC 구간(`v1.2.17-rc*`)에서 검증한 변경사항을 묶은 정식 릴리즈입니다.

### 핵심 변경 사항
- 드랍 파이프라인 정리:
  - `loot.currencyDropMode`는 `hybrid` 단일 정책으로 정리
  - 시체 드랍은 SPID 우선 + 런타임 폴백 구조로 안정화
- UserPatch 배포/사용성 개선:
  - 배포 ZIP에 UserPatch 위저드/EXE 포함
  - MO2 프로필 탐지/로드오더 기반 패치 생성 흐름 개선
  - UserPatch 결과 플러그인 ESL 플래그(ESPFE) 처리 지원
- MCM/런타임 설정 안정화:
  - 룬워드 조각/재련 오브 확률 슬라이더 및 런타임 반영 경로 개선
  - 외부 설정 파일 영속화 경로 보강
- 룬워드/툴팁 품질 개선:
  - 룬워드 효과/이름 표시 일관성 개선(한글/영문 혼재 완화)
  - 누락되던 설명 텍스트 보강 및 패널 반영 안정화
- 런타임 안정성/검증 체계 강화:
  - DoT 관측 set 하드캡 추가 등 메모리 안전성 보강
  - 루팅 핫패스 로그/조회 비용 정리
  - `lint_affixes.py`에 JSON Schema 검증(`--schema`) 추가 및 CI 반영

### 기본값/정책
- 기본 드랍 확률:
  - 룬워드 조각: `5%`
  - 재련 오브: `3%`
- 기본 드랍 모드: `hybrid`

### 검증
- `tools/release_verify.sh --fast`
- `CAFF_PACKAGE_VERSION=1.2.17 tools/build_mo2_zip.sh`

### 산출물
- `CalamityAffixes_MO2_v1.2.17_2026-02-21.zip`

### RC 요약본 (rc01~rc70)
- `rc01~rc18`:
  - 재련 오브 기반 재련 루프 정립, MCM 중복/소실 이슈 단계적 안정화, 상호작용 경로 CTD 가드 추가
- `rc19~rc30`:
  - 재련 오브 실체화/드랍 경로 정리, 프리뷰-실획득 불일치 완화, 획득 시 랜덤 어픽스 제거(재련 중심 정책 확립)
- `rc31~rc41`:
  - 룬워드 탭/툴팁 즉시 갱신 및 상태머신 보강, 동일 이름/인스턴스 키 정합성 개선, 런타임 모듈화 정리
- `rc42~rc51`:
  - 검증 계약(Contract) 강화, MCM 확률 런타임 브릿지/표시 포맷 수정, 룬워드 94종 한글 표기/설명 보강, LowHealth 트리거 추가
- `rc52~rc54`:
  - 드랍 시점 분리(시체·컨테이너 활성화 vs 월드 픽업), LVLI override 버그 수정, 패키징 스크립트 `--masters` 전달 안정화
- `rc55~rc63`:
  - UserPatch 도입/번들/위저드 UX 고도화, MO2 인식 개선, ESPFE 적용, 모드 몬스터 탐지 및 로드오더 충돌 복원 강화
- `rc64~rc70`:
  - SPID+런타임 하이브리드 정책 고정, 시체 런타임 과발동 정리, 확률 기본값 재조정(5%/3%), 메모리/핫패스/스키마 검증 체계 강화
